сбежа́ть -> to run down, run away
Verb, perfective, intransitive, stress pattern: b

Я сбегу́

Past: сбежа́л/сбежа́ла



Etymology: с- (s-) +‎ бежа́ть (bežátʹ)



Definitions (English):
  • to run down (from)
    • сбежа́ть с ле́стницы ― sbežátʹ s léstnicy ― to run downstairs, сбежа́ть по ле́стнице ― sbežátʹ po léstnice ― to run downstairs
  • to run away, to make off, to flee, to break out
    • to escape
      • to elope
