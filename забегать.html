забе́гать (6116)


Etymology 1[edit]

за- (za-) +‎ бегать (bégatʹ)

Pronunciation[edit]

  • IPA(key): [zɐˈbʲeɡətʲ]

Verb[edit]

забе́гать • (zabégatʹ) pf

  1. to start running
  2. to become shifty (of a look)

Etymology 2[edit]

за- (za-) +‎ бегать (bégatʹ) +‎ -ать (-átʹ)

Pronunciation[edit]

  • IPA(key): [zəbʲɪˈɡatʲ]

Verb[edit]

забе́гать • (zabegátʹ) impf (perfective забежа́ть)

  1. to get (в (v) + accusative or на (na) + accusative somewhere) by running or as if by running
    Он ка́ждый день забе́гает туда́ пе́ред рабо́той.
    On káždyj denʹ zabegájet tudá péred rabótoj.
    He runs that way every day before work.
  2. to run ahead (in anticipation), to run before someone
    Не надо забе́гать вперёд!
    Ne nádo zabegátʹ vperjód!
    No need to run ahead!
  3. (colloquial) to visit, to drop in (к (k) on)

забе́гать

Глаго́л, несоверше́нный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л соверше́нного ви́да — забежа́ть.

Приста́вка: за-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  1. передвига́ясь бе́гом, попа́дать куда-либо ◆ Он ка́ждый день забе́гает туда́ пе́ред рабо́той.
  2. передвига́ясь бе́гом, обгоня́ть или кого-либо, ока́зываться впереди́ или сбо́ку ◆ Не надо забе́гать вперёд!
