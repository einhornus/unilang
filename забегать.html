забе́гать -> to start rushing around
Verb, perfective, intransitive, stress pattern: a

Я забе́гаю
Ты забе́гаешь
Past: забе́гал/забе́гала



Etymologies:
  • за- (za-) +‎ бегать (bégatʹ)
  • за- (za-) +‎ бегать (bégatʹ) +‎ -ать (-átʹ)


Definitions (English):
  • to start running
    • to become shifty (of a look)
      • to get (
        • в (v) + accusative or на (na) + accusative somewhere) by running or as if by running. Он ка́ждый день забе́гает туда́ пе́ред рабо́той. On káždyj denʹ zabegájet tudá péred rabótoj. He runs that way every day before work.
      • to run ahead (in anticipation), to run before someone.
        • Не надо забе́гать вперёд! Ne nádo zabegátʹ vperjód! No need to run ahead!
      • (colloquial) to visit, to drop in (
        • к (k) on)
