забегать -> to start rushing around
Verb, perfective, intransitive, stress pattern: a

Я забегаю
Ты забегаешь
Past: забегал/забегала



Etymologies:
  • за- (za-) +‎ бе́гать (bégatʹ)
  • за- (za-) +‎ бе́гать (bégatʹ) +‎ -а́ть (-átʹ)


Definitions (English):
  • to start running
    • to become shifty (of a look)
      • to get (
        • в (v) + accusative or на (na) + accusative somewhere) by running or as if by running. Он ка́ждый день забега́ет туда́́ пе́ред рабо́то́й. On káždyj denʹ zabegájet tudá péred rabótoj. He runs that way every day before work.
      • to run ahead (in anticipation), to run before someone.
        • Не на́до забега́ть вперёд! Ne nádo zabegátʹ vperjód! No need to run ahead!
      • (colloquial) to visit, to drop in (
        • к (k) on)
