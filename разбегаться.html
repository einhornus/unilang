разбе́гаться -> to scatter, take a run-up
Verb, perfective, intransitive, stress pattern: a

Я разбе́гаюсь
Ты разбе́гаешься
Past: разбе́гался/разбе́галась



Etymologies:
  • раз- (raz-) +‎ бегать (bégatʹ) +‎ -ать (-átʹ) +‎ -ся (-sja)
  • раз- (raz-) +‎ бегать (bégatʹ) +‎ -ся (-sja)


Definitions (English):
  • to scatter,
    • разбе́гаться по места́м ― razbegátʹsja po mestám ― to run to one's places/posts/stations, глаза разбе́гаются у кого́-либо ― glazá razbegájutsja u kovó-libo ― one doesn't know where to look first
  • to take a run, to run up (before jumping, diving, etc)
    • (colloquial) to start running strenuously
      • (colloquial) to start running and not stop
