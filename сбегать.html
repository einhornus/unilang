сбега́ть (4554)


Etymology

с- (s-) +‎ бегать (bégatʹ) +‎ -ать (-átʹ)

Pronunciation 1

  • IPA(key): [zbʲɪˈɡatʲ]
  • Audio
    		(file)

Verb

сбега́ть • (sbegátʹ) impf (perfective сбежа́ть)

  1. to run down (from)
    сбега́ть с ле́стницы ― sbegátʹ s léstnicy ― to run downstairs
    сбега́ть по ле́стнице ― sbegátʹ po léstnice ― to run downstairs
  2. to run awaý, to make off, to flee, to break out
  3. to escape
  4. to elope

Pronunciation 2

  • IPA(key): [ˈzbʲeɡətʲ]
  • Audio
    		(file)

Verb

сбега́ть • (sbégatʹ) pf

  1. (colloquial) to run (to a place) and get back, to run (for), to go and get
    Сбегайте за врачо́м. ― Sbégajte za vračóm. ― Run for a doctor.
    Мне ну́жно сбега́ть в магази́н. ― Mne núžno sbégatʹ v magazín. ― I need to make a quick run over to the shop.
