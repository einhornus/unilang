сбегать -> to run down, run away
Verb, perfective, intransitive, stress pattern: a

Я сбегаю
Ты сбегаешь
Past: сбегал/сбегала



Etymology: с- (s-) +‎ бе́гать (bégatʹ) +‎ -а́ть (-átʹ)



Definitions (English):
  • to run down (from)
    • сбега́ть с ле́стницы ― sbegátʹ s léstnicy ― to run downstairs, сбега́ть по ле́стнице ― sbegátʹ po léstnice ― to run downstairs
  • to run away, to make off, to flee, to break out
    • to escape
      • to elope
        • (colloquial) to run (to a place) and get back, to run (for), to go and get.
          • Сбе́гайте за врачо́м. ― Sbégajte za vračóm. ― Run for a doctor. Мне ну́жно сбе́гать в магази́н. ― Mne núžno sbégatʹ v magazín. ― I need to make a quick run over to the shop.
