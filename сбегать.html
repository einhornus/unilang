сбега́ть (4556)


Etymology

с- (s-) + бе́гать (bégatʹ) + -а́ть (-átʹ)

Pronunciation 1

  • IPA(key): [zbʲɪˈɡatʲ]
  • Audio
    		(file)

Verb

сбега́ть • (sbegátʹ) impf (perfective сбежа́ть)

  1. to run down (from)
    сбега́ть с ле́стницы ― sbegátʹ s léstnicy ― to run downstairs
    сбега́ть по ле́стнице ― sbegátʹ po léstnice ― to run downstairs
  2. to run away, to make off, to flee, to break out
  3. to escape
  4. to elope

Pronunciation 2

  • IPA(key): [ˈzbʲeɡətʲ]
  • Audio
    		(file)

Verb

сбе́гать • (sbégatʹ) pf

  1. (colloquial) to run (to a place) and get back, to run (for), to go and get
    сбе́гайте за врачо́м. ― Sbégajte za vračóm. ― Run for a doctor.
    Мне ну́жно сбе́гать в магази́н. ― Mne núžno sbégatʹ v magazín. ― I need to make a quick run over to the shop.
