сбе́гать (4554)


Etymology

с- (s-) +‎ бегать (bégatʹ) +‎ -ать (-átʹ)

Pronunciation 1

  • IPA(key): [zbʲɪˈɡatʲ]
  • Audio
    		(file)

Verb

сбе́гать • (sbegátʹ) impf (perfective сбежа́ть)

  1. to run down (from)
    сбе́гать с ле́стницы ― sbegátʹ s léstnicy ― to run downstairs
    сбе́гать по ле́стнице ― sbegátʹ po léstnice ― to run downstairs
  2. to run away, to make off, to flee, to break out
  3. to escape
  4. to elope

Pronunciation 2

  • IPA(key): [ˈzbʲeɡətʲ]
  • Audio
    		(file)

Verb

сбе́гать • (sbégatʹ) pf

  1. (colloquial) to run (to a place) and get back, to run (for), to go and get
    Сбе́гайте за врачо́м. ― Sbégajte za vračóm. ― Run for a doctor.
    Мне ну́жно сбе́гать в магази́н. ― Mne núžno sbégatʹ v magazín. ― I need to make a quick run over to the shop.

сбе-гать

Глаго́л, несоверше́нный вид, непереходный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л соверше́нного ви́да — сбежа́ть.

Приста́вка: с-; ко́рень: -бег-; суффикс: -а; глагольное оконча́ние: -ть.

Значе́ние

  1. передвига́ясь бе́гом, спуска́ться
    Моня убра́л переговорник в карма́н и, выудив из-за па́зухи «беретту», ста́л сбе́гать по ле́стнице вниз.
  2. перен. бы́стро опуска́ться, стека́ть вниз
    Руче́й сбе́гает по скло́ну и исчеза́ет под сне́гом.
  3. соверша́ть побе́г
    А в основно́м, де́ти хотя́т жить свое́й жи́знью и ни от кого́ не зави́сеть, но все равно́ для э́того надо снача́ла поду́мать, а потом уже́ сбе́гать из дома.

Сино́нимы

  1. —
  2. низвергаться
