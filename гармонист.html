гармони́ст (24229)


Pronunciation

  • IPA(key): [ɡərmɐˈnʲist]

Noun

гармони́ст • (garmoníst) m anim (genitive гармони́ста, nominative plural гармони́сты, genitive plural гармони́стов)

  1. accordionist

Synonyms

  • аккордеони́ст (akkordeoníst), баяни́ст (bajaníst)

Related terms

  • гармо́ника (garmónika)
  • гармо́нь (garmónʹ)
  • гармо́шка (garmóška)

гар-мо-ни́ст

Существи́тельное, одушевлённое, мужско́й род, 2-е склонение (тип склоне́ния 1a по классифика́ции А. А. Зализняка).

Ко́рень: -гармон-; суффикс: -ист [Тихонов, 1996].

Значе́ние

  1. тот, кто игра́ет на гармо́ни или на гармо́шке
    Под гармо́шку веселе́й // Пе́сенки пою́тся! // То́лько плохо: гармони́сты // Больно задаю́тся! Часту́шка
    Поигра́й, гармони́ст, // Поигра́й, золото́й! // А я тебя́, гармони́ст, // Не оста́влю сирото́й! Часту́шка
    Гармони́сту за игру́, // За игру́ кипу́чую — // Два солёных огурца́, // Карто́шку рассыпучую! Часту́шка
    Гармони́ст пошёл на низ, — // Мы ему́ покланялись! // На́ши ни́зкие покло́ны // Ему́ не понра́вились! Часту́шка
  2. музыка́нт, игра́ющий на гармо́ни
    Так прие́хал в Гро́зный превосхо́дный пандурист Султан Самбиев, отли́чный скрипа́ч Джамалдин Абиев, а впоследствии, когда́ в орке́стр бы́ли введены́ гармо́ники, и даровитый гармони́ст Димаев. В. Энке, В. Волков, «Чечено-ингушский анса́мбль» // «Наро́дное тво́рчество», 1937 г.
