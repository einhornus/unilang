Noun[edit]

гармони́ст • (garmoníst) m anim (genitive гармони́ста, nominative plural гармони́сты, genitive plural гармони́стов)

  1. accordionist

Synonyms[edit]

  • аккордеонист (akkordeoníst), баяни́ст (bajaníst)

Related terms[edit]

  • гармо́ника (garmónika)
  • гармо́нь (garmónʹ)
  • гармо́шка (garmóška)

Значе́ние[пра́вить]

  1. тот, кто игра́ет на гармо́ни или на гармо́шке ◆ Под гармо́шку веселе́й // Пе́сенки пою́тся! // То́лько пло́хо: гармони́сты // Больно задаю́тся! Часту́шка ◆ Поигра́й, гармони́ст, // Поигра́й, золото́й! // А я тебя́, гармони́ст, // Не оста́влю сирото́й! Часту́шка ◆ Гармони́сту за игру́, // За игру́ кипу́чую — // Два солёных огурца́, // Карто́шку рассыпучую! Часту́шка ◆ Гармони́ст пошёл на низ, — // Мы ему́ покланялись! // На́ши ни́зкие покло́ны // Ему́ не понра́вились! Часту́шка
  2. музыка́нт, игра́ющий на гармо́ни ◆ Так прие́хал в Гро́зный превосхо́дный пандурист Султа́н Самбиев, отли́чный скрипа́ч Джамалдин Абиев, а впоследствии, когда́ в орке́стр бы́ли введены гармо́ники, и даровитый гармони́ст Димаев. В. Энке, В. Волко́в, «Чечено-ингушский анса́мбль» // «Наро́дное тво́рчество», 1937 г. (цита́та из Национа́льного корпуса ру́сского языка́, см. Спи́сок литерату́ры)
