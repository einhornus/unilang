Значение

  1. ве́рхний слой коры берёзы
    В дре́вней Ру́си на бересте писали, сухо́й перегонкой из неё добыва́ли дёготь, из бересты де́лали дома́шнюю у́тварь. Клуб «Ю́ный хи́мик» // «Хи́мия и жизнь», 1968 г. (цита́та из Национа́льный ко́рпус русского языка">Национального корпуса русского языка, см. Спи́сок литерату́ры)
    Из содранной с берёзовых стволо́в бересты гна́ли дёготь для сма́зки теле́г и сапо́г, для други́х на́добностей, из бересты мастери́ли туески для воды, для сбо́ра лесны́х я́год, шла́ береста на подсти́лку тесовой кро́вли. С. М. Голицын, «Сказа́ния о земле́ Моско́вской», 1968–1988 гг. (цита́та из Национа́льный ко́рпус русского языка">Национального корпуса русского языка, см. Спи́сок литерату́ры)
