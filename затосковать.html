затоскова́ть (20593)


Etymology

за- (za-) +‎ тоскова́ть (toskovátʹ)

Pronunciation

  • IPA(key): [zətəskɐˈvatʲ]

Verb

затоскова́ть • (zatoskovátʹ) pf (по (po) + dative)

  1. (colloquial) to start to be sad́
  2. (colloquial) to start to be bored
  3. (colloquial) to start to miss, to start to yearn for
  4. (colloquial) to start to plaintively express longing for

Related terms

  • тоскова́ть (toskovátʹ), тоскование (toskovánije)
  • тоска́ (toská)
  • тоскли́вый (tosklívyj), тоскливо (tosklívo), тоскливость (tosklívostʹ)
