пробежа́ть -> to run (past/through/a certain distance)
Verb, perfective, transitive, stress pattern: b

Я пробегу
Ты пробежи́шь
Past: пробежа́л/пробежа́ла



Etymology: про- (pro-) +‎ бежа́ть (bežátʹ)



Definitions (English):
  • to run past, to run through
    • to run a certain distance or for some time
      • to skim, to scan, to look through
