Значе́ние[пра́вить]

  1. свя́занный, соотнося́щийся по значе́нию с существи́тельным водоро́д ◆ Отсу́тствует приме́р употребле́ния (см. ).
  2. сво́йственный водоро́ду, характе́рный для него́ ◆ Отсу́тствует приме́р употребле́ния (см. ).
  3. содержа́щий в своём хими́ческом соста́ве водоро́д ◆ Отсу́тствует приме́р употребле́ния (см. ).
  4. испо́льзующий водоро́д как термоя́дерное то́пливо ◆ Отсу́тствует приме́р употребле́ния (см. ).
  5. производи́мый с по́мощью водоро́да ◆ Отсу́тствует приме́р употребле́ния (см. ).
  6. рожда́ющий во́ду ◆ Орле преславный, высоко па́рящий, // Сла́вою орлы вся́ превосходя́щий, // Вскую превы́ше облак водоро́дных // Париши крилы перий славоплодных? Симеон Полоцкий, «Орел российский», 1667 г.
  7. рождённый в воде́, из воды ◆ Чу́да морски́е еще и страши́лища все водоро́дны // Бездн из вла́жных свои́х и глубо́ких те́х исходи́ли... В. К. Тредиаковский, «Тилемахида», 1766 г.
