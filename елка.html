ёл-ка

Существи́тельное, неодушевлённое, же́нский род, 1-е склонение (тип склоне́ния 3*a по классифика́ции А. А. Зализняка).

Ко́рень: -ёл-; суффикс: -к; оконча́ние: -а [Тихонов, 1996].

Значе́ние

  1. ботан., разг. ель, хво́йное дерево с коро́ткими иго́лками
    Одино́ко выраста́ла // Ёлка стро́йная в ле́су, — // Хо́лод смо́лоду узна́ла, // Ча́сто ви́дела грозу́. И. С. Никитин, «Ёлка»
  2. перен. нового́днее пра́зднество для дете́й
    Ёлку хотя́т устро́ить? Пусть! Картонажами её уве́шать хотя́т — пусть забавля́ются. А. Т. Аверченко, «Детвора́»
