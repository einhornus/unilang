папка -> file, folder
Noun, declension 1a (pattern девочка)

Plural: папки
Genitive: папки



Etymologies:
  • па́па (pápa, “dad”) +‎ -ка (-ka).
  • Borrowed from German Pappe (“cardboard”) +‎ -ка (-ka).


Definitions (English):
  • daddy
    • folder (an organizer for storing papers)
      • file (things stored in a folder)
        • (computing) folder (directory or virtual container in a file system)
