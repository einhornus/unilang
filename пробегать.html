Verb, perfective, transitive, stress pattern: a

Etymologies:

  • про- (pro-) +‎ бегать (bégatʹ)


Definitions (English):
  • to run past, to run through
    • to run a certain distance or for some time
      • to skim, to scan, to look through
        • to spend some time running
