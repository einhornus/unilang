обежа́ть (17135)


Etymology[edit]

о- (o-) +‎ бежа́ть (bežátʹ)

Pronunciation[edit]

  • IPA(key): [ɐbʲɪˈʐatʲ]
  • Homophone: обижа́ть (obižátʹ)

Verb[edit]

обежа́ть • (obežátʹ) pf (imperfective обе́гать)

  1. to run around (+ accusative or вокру́г (vokrúg))
  2. to run past (+ ми́мо (mímo))
  3. (colloquial) to run (everywhere or many places)
  4. (colloquial) to drop by (everywhere or many places)

Related terms[edit]

  • бежа́ть impf (bežátʹ), побежа́ть pf (pobežátʹ)
  • бегать impf (bégatʹ), побе́гать pf (pobégatʹ)
  • бег (beg)
  • бе́гом (begóm)
  • бегле́ц (begléc), беглянка (begljánka)
  • бе́глый (béglyj), бе́гло (béglo), беглость (béglostʹ)
  • беготня́ (begotnjá)
  • бе́гство (bégstvo)
  • бе́женец (béženec), беженка (béženka)

Глаго́л, совершенный вид, перехо́дный, тип спряжения по классифика́ции А. Зализняка — 5b^. Соотве́тствующий глаго́л несоверше́нного ви́да — обе́гать.

Приста́вка: о-; ко́рень: -беж-; суффикс: -а; глагольное оконча́ние: -ть [Тихонов, 1996].

Значе́ние[пра́вить]

  1. бе́гая, перемести́ться вокру́г кого-либо или чего-либо ◆ Отсу́тствует приме́р употребле́ния (см. ).
  2. пробежа́ть стороно́й, окружны́м путём ◆ Отсу́тствует приме́р употребле́ния (см. ).
  3. , торопли́во переходя́ из одного́ места в друго́е, посети́ть мно́гих или многие места; на́скоро обойти́ кого-либо или что-либо ◆ Дву́х дельфи́нов тут призва́ли // И, отда́в ука́з, сказа́ли, // Чтоб, от и́мени царя́, // Обежа́ли все моря // И того́ Ерша-гуляку, // Крикуна́ и забияку, // Где бы ни бы́ло, нашли́, // К госуда́рю привели́. П. П. Ершов, «Конёк-Горбунок», 1834 г.
