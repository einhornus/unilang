резви́ться (16227)


Etymology

ре́звый (rézvyj) +‎ -ить (-ítʹ) +‎ -ся (-sja)

Pronunciation

  • IPA(key): [rʲɪzˈvʲit͡sːə]

Verb

резви́ться • (rezvítʹsja) impf

  1. to gambol, to frisk, to romp

Usage notes

Accent pattern c (резви́шься, резви́тся, резвя́тся, etc.) was found in the older language[1][2]:

1800, Gavrila Derzhavin, <cite>У́тро:
Там, вы́плыв из пучи́ны, // Игра́ют, резвя́тся дельфи́ны, // И рыб стада сверка́ют чешуёй, // И бле́щут чуды чре́ва белизно́й.
Tam, výplyv iz pučíny, // Igrájut, rézvjatsja delʹfíny, // I ryb stadá sverkájut češujój, // I bléščut čudy čréva beliznój.
