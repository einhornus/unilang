иссяка́ть (21456)


Etymology

исся́кнуть (issjáknutʹ) +‎ -ать (-átʹ)

Pronunciation

  • IPA(key): [ɪsʲːɪˈkatʲ]

Verb

иссяка́ть • (issjakátʹ) impf (perfective исся́кнуть)

  1. to run dry, to dry up
  2. to run out, to disappear, to fail, to run low, to run short
    моё терпе́ние иссяка́ет ― mojó terpénije issjakájet ― my patience is running out (is wearing thin)

Related terms

  • сякнуть (sjáknutʹ)

Глаго́л, несоверше́нный вид, перехо́дный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л соверше́нного ви́да — исся́кнуть.

Ко́рень: -иссяк-; суффикс: -а; глагольное оконча́ние: -ть [Тихонов, 1996].

Значение

  1. высыха́ть, лиша́ться воды, вла́ги
  2. перен. постепе́нно зака́нчиваться, исче́рпываться

Синонимы

  1. высыха́ть, пересыхать
  2. зака́нчиваться, исче́рпываться
