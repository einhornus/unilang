спра́вочный (14134)


Etymology

спра́вка (správka) + -ный (-nyj)

Pronunciation

  • IPA(key): [ˈspravət͡ɕnɨj]

Adjective

спра́вочный • (správočnyj)

  1. (relational) inquiry
    спра́вочное бюро́ ― správočnoje bjuró ― inquiry office; information desk
    спра́вочное изда́ние; спра́вочная кни́га ― správočnoje izdánije; správočnaja kníga ― reference book
  2. (relational, engineering, computing) reference
    спра́вочный разме́р ― správočnyj razmér ― reference size
    спра́вочный файл ― správočnyj fajl ― help file
    спра́вочные да́нные ― správočnyje dánnyje ― reference data
  3. (relational) benchmark
    спра́вочная цена́ ― správočnaja cená ― benchmark price

Related terms

  • спра́вка (správka)
  • справля́ть impf (spravljátʹ), спра́вить pf (správitʹ)
  • справля́ться impf (spravljátʹsja), спра́виться pf (správitʹsja)
  • спра́вочник (správočnik)

Прилага́тельное, тип склоне́ния по классифика́ции А. Зализняка — 1a.

  1. свя́занный, соотнося́щийся по значе́нию с существи́тельным спра́вка; содержа́щий в себе́ спра́вки
