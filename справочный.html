Adjective[edit]

спра́вочный • (správočnyj)

  1. (relational) inquiry
    спра́вочное бюро́ ― správočnoje bjuró ― inquiry office; information desk
    спра́вочное изда́ние; спра́вочная кни́га ― správočnoje izdánije; správočnaja kníga ― reference book
  2. (relational, engineering, computing) reference
    спра́вочный размер ― správočnyj razmér ― reference size
    спра́вочный файл ― správočnyj fajl ― help file
    спра́вочные да́нные ― správočnyje dánnyje ― reference data
  3. (relational) benchmark
    спра́вочная цена ― správočnaja cená ― benchmark price
