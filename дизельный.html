Прилага́тельное, тип склоне́ния по классифика́ции А. Зализняка — 1*a.

Ко́рень: -дизель-; суффикс: -н; оконча́ние: -ый [Тихонов, 1996].

Значе́ние

  1. техн. рабо́тающий по при́нципу поршневого дви́гателя вну́треннего сгора́ния с воспламенением то́плива от сжа́тия
    Привод― то́лько за́дний, моторы― ди́зельные, объёмом 3, 0 и 3, 3 л. Анатолий Карпенков, Юрий Нечетов, «Балти́йские головастики», 2003 г. // «За рулём»
  2. техн. автомоб. оснащённый ди́зельным [1] дви́гателем
    На пра́ктике э́то хорошо заме́тно на ско́льзком покры́тии, когда́ ди́зельная Volvo да́же при ре́зком уда́ре по педа́ли в спорти́вном режи́ме рабо́ты трансми́ссии не выдаёт ни еди́ного скри́па покры́шек. Владимир Гаврилов, Тест-драйв Volvo S60 г. // «2011»
  3. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (силова́я устано́вка с ди́зельным дви́гателем)
  4. про́ст. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (дизель-поезд)
  5. про́ст. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (ди́зельное то́пливо)
