Значе́ние[пра́вить]

  1. рабо́тающий по при́нципу поршневого дви́гателя вну́треннего сгора́ния с воспламенением то́плива от сжа́тия ◆ Привод― то́лько за́дний, моторы― ди́зельные, объёмом 3, 0 и 3, 3 л. Анатолий Карпенков, Юрий Нечетов, «Балти́йские головастики», 2003 г. // «За рулём» (цита́та из Национа́льного корпуса ру́сского языка́, см. Спи́сок литерату́ры)
  2. оснащённый ди́зельным [1] дви́гателем ◆ На пра́ктике э́то хорошо́ заме́тно на ско́льзком покры́тии, когда́ ди́зельная Volvo да́же при ре́зком уда́ре по педа́ли в спорти́вном режи́ме рабо́ты трансми́ссии не выдаёт ни еди́ного скри́па покры́шек. Владимир Гаврилов, Тест-драйв Volvo S60 г. // «2011» (цита́та из Национа́льного корпуса ру́сского языка́, см. Спи́сок литерату́ры)
  3. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (силова́я устано́вка с ди́зельным дви́гателем) ◆ Отсу́тствует приме́р употребле́ния (см. ).
  4. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (дизель-поезд) ◆ Отсу́тствует приме́р употребле́ния (см. ).
  5. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (ди́зельное то́пливо) ◆ Отсу́тствует приме́р употребле́ния (см. ).
