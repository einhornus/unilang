дви-га-ться

Глаго́л, несоверше́нный вид, непереходный, возвра́тный, тип спряжения по классифика́ции А. Зализняка — 1a. Соотве́тствующий глаго́л соверше́нного ви́да — дви́нуться. В значе́нии тро́гаться с места употребля́ется спряжение то́лько по схе́ме 1a.

Ко́рень: -двиг-; суффикс: -а; глагольное оконча́ние: -ть; постфикс: -ся [Тихонов, 1996].

Значение

  1. находи́ться в движе́нии, перемеща́ться, изменя́ть своё положе́ние в простра́нстве
    Колёса дви́жутся водой.
    Земля́ дви́жется вокру́г свое́й оси.
    Нам хоте́лось бы, подо́бно Иисусу Навину, сказа́ть: стой, со́лнце, не дви́жься! М. Е. Салтыков-Щедрин, «Статьи́ из «Совреме́нника»», 1863 г. (цита́та из Национа́льный ко́рпус русского языка">Национального корпуса русского языка, см. Спи́сок литерату́ры)
  2. направля́ться, идти́ вперёд,
    Толпа́ ме́дленно дви́галась по у́лице.
    Парохо́д дви́жется вверх по Волге.
  3. разг. фамилья́рное">фам. тро́гаться с места, отправля́ться в путь
    Мы ско́ро дви́гаемся на да́чу.
  4. прикла́дывать физи́ческие уси́лия
    Врач рекомендова́л мне больше дви́гаться.
  5. разг. получа́ть повыше́ния по слу́жбе, де́лать служе́бную карье́ру
    Бы́стро дви́гаться по слу́жбе.

Синонимы

  1. перемеща́ться, передвига́ться
  2. направля́ться
  3. тро́гаться, отправля́ться
  4. -
  5. расти́
