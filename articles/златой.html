Adjective

злато́й • (zlatój)

  1. (archaic, poetic) gold, golden
|||||

Значе́ние

  1. устар. свя́занный, соотнося́щийся по значе́нию с существи́тельным злато
  2. устар. сво́йственный зла́ту, характерный для него́
  3. устар., поэт. сде́ланный или состоя́щий из зо́лота; золото́й
    • Фарао́н, седя на златом престо́ле, окру́жен был знатнейшими двора́ своего́; чело́ его́ увенчано бы́ло блистающим венцо́м, ски́петр держа́л он в свое́й десни́це.
    • У лукоморья дуб зелёный; // Злата́я цепь на ду́бе том: // И днём и но́чью кот учёный // все хо́дит по це́пи кругом;
    • Злато́й вено́к на голове́ притво́рства, // И де́ва-скро́мность здесь осквернена.
  4. устар. позоло́ченный
    • Сто́я на сей горе, ви́дишь на пра́вой стороне́ почти́ всю Москву, сию́ ужа́сную грома́ду домо́в и церкве́й, кото́рая представля́ется глаза́м в о́бразе вели́чественного амфитеа́тра: великоле́пная карти́на, особли́во когда́ све́тит на неё со́лнце, когда́ вече́рние лучи́ его́ пыла́ют на бесчи́сленных златы́х купола́х, на бесчи́сленных креста́х, к не́бу возносящихся!
  5. устар., поэт. цве́том напомина́ющий зо́лото
    • Я помню го́лос ми́лых слов, // Я помню о́чи голубы́е, // Я помню ло́коны златы́е // Небре́жно вью́щихся власов.
    • Меж нив златы́х и пажитей зелёных // оно, сине́я, сте́лется широко;
    • На темно́-голубо́м эфи́ре // Злата́я пла́вала луна́
  6. устар., перен., поэт. блаже́нный, счастли́вый
    • Исполнен мы́слями златы́ми, // Не понима́емый нике́м, // Пе́ред распу́тьями земны́ми // Прохо́дишь ты, уны́л и нем.
    • Куда́, куда́ вы удали́лись, // весны мое́й златы́е дни?
