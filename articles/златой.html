злато́й (25806)


Etymology

зла́то (zláto) + -ой (-oj)

Pronunciation

  • IPA(key): [zlɐˈtoj]

Adjective

злато́й • (zlatój)

  1. (archaic, poetic) gold, golden

Прилага́тельное, ка́чественное, тип склоне́ния по классифика́ции А. Зализняка — 1b⊠. Сравни́тельная сте́пень — злате́е, злате́й, позлате́е, позлате́й. Кра́ткая фо́рма муж. р. не испо́льзуется, образова́ние остальны́х кра́тких фо́рм затрудни́тельно.

Ко́рень: -злат-; оконча́ние: -ой [Тихонов, 1996].

Значе́ние

  1. устар. свя́занный, соотнося́щийся по значе́нию с существи́тельным злато
  2. устар. сво́йственный зла́ту, характерный для него́
  3. устар., поэ́т. сде́ланный или состоя́щий из зо́лота; золото́й
    Фарао́н, седя на златом престо́ле, окружен был знатнейшими двора́ своего́; чело́ его́ уве́нчано бы́ло блиста́ющим венцо́м, ски́петр держа́л он в свое́й десни́це. Д. И. Фонвизин, «Иосиф», 1769 г.
    У лукоморья дуб зелёный; // Злата́я цепь на ду́бе том: // И днём и но́чью кот учёный // Все хо́дит по це́пи кругом; А. С. Пушкин, «Руслан и Людмила», 1836 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
    Злато́й венок на голове́ притво́рства, // И дева-скромность здесь осквернена́. У. Шекспир, «Соне́т 66» / перево́д А. В. Луначарского, 1609 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
  4. устар. позоло́ченный
    Сто́я на сей горе, ви́дишь на пра́вой стороне́ почти́ всю Москву, сию́ ужа́сную грома́ду домо́в и церкве́й, кото́рая представля́ется глаза́м в о́бразе вели́чественного амфитеа́тра: великоле́пная карти́на, особли́во когда́ све́тит на неё со́лнце, когда́ вече́рние лучи́ его́ пыла́ют на бесчи́сленных златы́х купола́х, на бесчи́сленных креста́х, к не́бу возносящихся! Н. М. Карамзин, «Бе́дная Лиза», 1792 г.
  5. устар., поэ́т. цве́том напомина́ющий зо́лото
    Я помню го́лос ми́лых сло́в, // Я помню о́чи голубы́е, // Я помню ло́коны златы́е // Небре́жно вью́щихся власов. К. Н. Батюшков, «Мой ге́ний», 1815 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
    Меж ни́в златы́х и пажитей зелёных // Оно, сине́я, сте́лется широко; А. С. Пушкин, «Вновь я посети́л…», 1835 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
    На темно-голубом эфи́ре // Злата́я пла́вала луна́ Г. Р. Державин, «Видение Мурзы́», 1783—1784 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
  6. устар., перен., поэ́т. блаже́нный, счастли́вый
    Испо́лнен мы́слями златы́ми, // Не понима́емый нике́м, // Пе́ред распу́тьями земны́ми // Прохо́дишь ты, уны́л и нем. А. С. Пушкин, «Езерский», 1836 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
    Куда́, куда́ вы удали́лись, // Весны мое́й златы́е дни? А. С. Пушкин, «Евгений Онегин», 1830 г. (цита́та из Библиоте́ки Максима Мошкова, см. Спи́сок литерату́ры)
