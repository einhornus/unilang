Значе́ние

  1. свя́занный, соотнося́щийся по значе́нию с существи́тельным фаянс
  2. сво́йственный фаянсу, характерный для него́, похо́жий на фаянс
  3. сде́ланный, изгото́вленный из фаянса
    • Прозра́чная голуба́я глазурь покрыва́ла фая́нсовые плиты на стенах ко́мнат, сквозь неё не́жно мерца́ли, просве́чивая, замыслова́тые рису́нки из листово́го зо́лота ― волше́бное произведе́ние иску́сства.
  4. предназначенный для изготовле́ния фаянса
