со-ли́ть

Глаго́л, несовершенный вид, перехо́дный, тип спряжения по классификации А. Зализняка — 4c. Соотве́тствующие глаго́лы соверше́нного ви́да — приставочные ти́па посоли́ть, засолить. Встреча́ется та́кже вариа́нт спряжения по схе́ме 4b: фо́рмы наст. вр. — соли́шь, соли́т, соли́м, соли́те, соля́т.

Значе́ние

  1. посыпа́ть, добавля́ть пова́ренную соль (в пищу)
    • ― солите, солите, ба́тюшка, ― сказа́ла хозя́йка, придвига́я соло́нку.
    • потом сыр формуют, прессу́ют, солят и оставля́ют созрева́ть.
    • Мы хлеб соли́ли кру́пной солью, // и на ходу, легко́ дыша́, // мы с э́тим хле́бом е́ли со́ю // и пили во́ду из ковша́.
  2. консервировать, заготовлять впрок с солью, в солёном раство́ре
    • она езжа́ла по рабо́там, // Соли́ла на зи́му грибы́, // Вела́ расхо́ды, бри́ла лбы, // Ходи́ла в ба́ню по суббо́там…
    • он вари́л варе́нье и соли́л на зи́му огурцы́.
  3. перен., разг. вреди́ть
    • ― Бре́шет Степка! Э́то он за Гришку мне солит.
