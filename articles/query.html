query (3309)


ед. ч. мн. ч.
query queries

Существи́тельное

Значе́ние

  1. вопро́с
    • The teacher answered the student's query concerning biosynthesis. — Учи́тель отве́тил на вопро́с студе́нта, свя́занный с биосинтезом.
  2. запро́с (в какую-л. инста́нцию)
  3. комп. запро́с
    • The database admin switched on query logging for debugging purposes. — Админ ба́з да́нных включи́л сохране́ние запро́сов в це́лях отладки.
  4. сомне́ние
  5. вопроси́тельный знак

Сино́нимы

  1. -
  2. -

Значе́ние

  1. спра́шивать (if, whether); осведомля́ться; выясня́ть, уточня́ть; консульти́роваться
  2. запра́шивать (информа́цию и т. п.)
  3. выража́ть сомне́ние, подверга́ть сомне́нию (about; as to)
  4. помечать вопроси́тельным знаком
