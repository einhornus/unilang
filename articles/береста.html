Значе́ние

  1. ве́рхний слой коры берёзы
    • В дре́вней руси на бересте́ писали, сухо́й перегонкой из неё добыва́ли дёготь, из бересты́ де́лали дома́шнюю у́тварь.
    • Из содранной с берёзовых стволо́в бересты́ гна́ли дёготь для сма́зки теле́г и сапо́г, для други́х на́добностей, из бересты́ мастери́ли туески для воды, для сбо́ра лесны́х я́год, шла береста́ на подсти́лку тесо́вой кро́вли.
