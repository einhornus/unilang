Приста́вка: с-; ко́рень: -чёт-; суффикс: -н; оконча́ние: -ый [Тихонов, 1996].

Значе́ние

  1. предназначенный, слу́жащий для счёта, подсчитывания, вычисле́ний
    МЭСМ (ма́лая электро́нная счётная маши́на) была пе́рвой оте́чественной универса́льной ламповой ЭВМ в Ссср.
  2. счетоводный
    В осно́ве формирова́ния счётных пала́т лежи́т законода́тельство субъе́ктов РФ.
  3. принадлежа́щий счётам
    Кишенский уда́рил гро́мко счётною костя́шкой и отвеча́л… Н. С. Леско́в, «На ножа́х», 1870 г.
  4. подсчи́танный, учтённый
    Раз в крещенский вечеро́к // Де́вушки гада́ли: // За во́рота башмачо́к, // Сня́в с ноги, броса́ли; // Снег поло́ли; под окно́м // Слу́шали; корми́ли // Счётным ку́рицу зерно́м; // Я́рый воск топили. В. А. Жуковский, «Светлана»
    Не для меня́ ме́рные ла́ски, не для меня́ счётные поцелу́и. А. А. Бестужев-Марлинский, «Он был уби́т», 1835-1836 г.
    В после́днюю кампа́нию неприя́тель потеря́л счётных се́мьдесят пять ты́сяч, то́лько что не сто ты́сяч. А. В. Суворов, «Нау́ка побежда́ть», 1795 г.
  5. матем. поддаю́щийся исчисле́нию, исчислимый
    Други́ми словами, счётное мно́жество — э́то мно́жество, равномощное мно́жеству натура́льных чи́сел.

Сино́нимы

  1. вычисли́тельный
  2. счетоводный
  3. исчислимый
