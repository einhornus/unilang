при-го-то́в-лен-ный

Страдательное прича́стие, совершенного ви́да, проше́дшего вре́мени, тип склоне́ния по классифика́ции А. Зализняка — 1*a(2). Соотве́тствующее прича́стие несоверше́нного ви́да проше́дшего вре́мени — приготовля́емый.

Приста́вка: при-; ко́рень: -готовл-; суффикс: -енн; оконча́ние: -ый [Тихонов, 1996].

Значе́ние

  1. страд. прич. прош. вр. от́ пригото́вить; го́дный, гото́вый к чему-нибудь, приго́дный для испо́льзования
    Обе́денный стол был накрыт крахма́льной ска́тертью и пригото́влен для вече́рнего ча́я — попо́вские ча́шки, корзи́нка с бисквитами, лимо́н, торт, золочёные вилочки, таре́лочки. В. П. Катаев, «Алма́зный мой венец», 1975—1977 г.
    Ко вре́мени вы́пуска Любаши из образцо́вого пансио́на была пригото́влена для неё, по общезаведённому у нас поря́дку, го́рничная, молода́я и дово́льно приго́жая де́вушка, взя́тая наро́чно для э́того поко́йницей Анной Алексеевной во двор. В. И. Даль, «Павел Алексеевич Игри́вый», 1847 г.
  2. специально со́зданный, разрабо́танный для чего-либо
    Там для вас бу́дет пригото́влен специа́льный курс по интересу́ющей тема́тике. А. М. Тарасов, «Миллионе́р», 2004 г.
  3. подгото́вленный для приёма в пищу; сва́ренный, сжаренный, спечённый
