служа́нка (13319)


Pronunciation

  • IPA(key): [slʊˈʐankə]

Noun

служа́нка • (služánka) f anim (genitive служа́нки, nominative plural служа́нки, genitive plural служа́нок, masculine слуга́)

  1. maidservant, housemaid

Related terms

  • слуга́ (slugá)

Существи́тельное

Значе́ние

  1. устар. же́нская прислу́га, дома́шняя рабо́тница; же́нщина, выполня́ющая хозя́йственные услу́ги по дому
    • Она езжа́ла по рабо́там, // Соли́ла на зи́му грибы́, // Вела́ расхо́ды, бри́ла лбы́, // Ходи́ла в ба́ню по суббо́там, // Служа́нок би́ла осердясь — // Все э́то му́жа не спросясь.
    • Господа отку́шали, — отвеча́ла служа́нка; встаю́т изо́ стола́.
    • Служа́нка несла́ за не́ю два узла́.
  2. книжн., перен. что-либо име́ющее лишь подсо́бное, служе́бное значе́ние для чего-либо бо́лее ва́жного, зна́чимого и т. п.
    • Филосо́фия до́лго была служа́нкой теоло́гии.

Сино́нимы

  1. прислужница, прислу́га
