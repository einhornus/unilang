забежа́ть (8473)


Etymology

за- (za-) + бежа́ть (bežátʹ)

Pronunciation

  • IPA(key): [zəbʲɪˈʐatʲ]

Verb

забежа́ть • (zabežátʹ) pf (imperfective забега́ть)

  1. to get (в (v) + accusative or на (na) + accusative somewhere) by running or as if by running
    он забежа́л туда́ пе́ред рабо́той.
    On zabežál tudá péred rabótoj.
    He ran that way before work.
  2. to run ahead (in anticipation), to run before someone
    забежа́ть вперёд ― zabežátʹ vperjód ― to run ahead; to anticipate
  3. (colloquial) to visit, to drop in (к (k) on)
