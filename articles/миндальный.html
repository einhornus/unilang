минда́льный (26399)

|||||||||||||||

Значе́ние

  1. свя́занный, соотнося́щийся по значе́нию с существи́тельным минда́ль (де́рево)
  2. состоя́щий из минда́льных дере́вьев
  3. свя́занный, соотнося́щийся по значе́нию с существи́тельным минда́ль (оре́х)
  4. добыва́емый, получа́емый из миндаля́
  5. пригото́вленный из миндаля́, с миндалём
  6. ро́зовый, как цветки́ миндаля́
  7. перен., разг. преувеличенно не́жный, слаща́вый
