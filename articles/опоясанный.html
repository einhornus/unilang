опоя́санный (24311)


Pronunciation

  • IPA(key): [ɐpɐˈjasən(ː)ɨj]

Participle

опоя́санный • (opojásannyj)

  1. past passive perfective participle of опоя́сать (opojásatʹ)

Значе́ние

  1. страд. прич. прош. вр. от́ опоя́сать
  2. перен., поэ́т. быть по́лным чего-либо
    • И, опоя́санная сла́вой, // Отражена в игре́ крова́вой // Живы́ми и́грами штыко́в.
    • В ри́зе свя́тости и сла́вы, // Опоя́сан стариной, // Ста́рец — Киёв предо мной // Возблистал золотоглавый.
