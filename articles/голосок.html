Etymology

го́лос (gólos) + -о́к (-ók)

$!$$

Noun

голосо́к • (golosók) m inan (genitive голоска́, nominative plural голоски́, genitive plural голоско́в)

  1. diminutive of го́лос (gólos): (high or soft) voice
|||||

Значе́ние

  1. ласк. к го́лос
    • Раздайсь, голосо́к, // По́ бору сыро́му! // Ох, донеси́сь, голосо́к, // К моёму дорого́му!
    • Невесёлая неделюшка, — // Не ви́жу я дружка! // Не игра́ет во тальяночку, // Не слы́шно голоска́!
    • Говори́ла голоску́: // «Разда́йся, го́лос, по леску́, // Чтобы ми́лого-краси́вого // Уда́рило в тоску́!»
  2. уменьш. к го́лос; ти́хий, сла́бый го́лос
    • Голосо́к у неё был то́ненький-то́ненький, сла́бый-сла́бый: как у прищемлённой пти́чки.

Сино́нимы

  1. голосочек
  2. голосишко
