Значе́ние

  1. техн. рабо́тающий по при́нципу поршневого дви́гателя вну́треннего сгора́ния с воспламенением то́плива от сжа́тия
    • Привод― то́лько за́дний, моторы― ди́зельные, объёмом 3, 0 и 3, 3 л.
  2. техн. автомоб. оснащённый ди́зельным [1] дви́гателем
    • На пра́ктике э́то хорошо́ заме́тно на ско́льзком покры́тии, когда́ ди́зельная Volvo да́же при ре́зком уда́ре по педа́ли в спорти́вном режи́ме рабо́ты трансми́ссии не выдаёт ни еди́ного скри́па покры́шек.
  3. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (силова́я устано́вка с ди́зельным дви́гателем)
  4. прост. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (дизель-по́езд)
  5. прост. свя́занный, соотнося́щийся по значе́нию с существи́тельным дизель (ди́зельное то́пливо)
