си́речь (27521)


Сою́з

Значе́ние

  1. устар. то есть, ина́че/точнее говоря́, (а) именно
    • Царь так начал речь: «Послу́шай, // На тебя́ доно́с, Ванюша. // Говоря́т, что вот сейча́с // Похваля́лся ты для нас // Отыска́ть другу́ю пти́цу, // Си́речь мо́лвить, Царь-девицу…»

Сино́нимы

  1. диал. силечь, сильчь, то есть, то бишь, сло́вом
