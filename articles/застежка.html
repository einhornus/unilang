Значе́ние

  1. приспособле́ние для застегивания оде́жды, о́буви
    • Мада́м Зина оде́та кри́во, ко́со, ю́бка из ры́жего дра́па спе́реди разорвана, застёжка на блу́зе без басонных пу́говиц — позо́р фо́рменный!
  2. приспособле́ние для скрепления сторо́нок перёплета кни́ги
    • старик, закры́в святу́ю кни́гу, // Застёжки ме́дные сомкну́л.
