по́льзовать (25442)

[ˈpolʲzəvətʲ]
|||||

по́льза (pólʹza) + -овать (-ovatʹ)


|||||

Verb

по́льзовать • (pólʹzovatʹ) impf (perfective попо́льзовать)

  1. (literary or dated) to benefit
  2. to use, to exploit
  3. (literary or dated) to treat (of a doctor and patient)

Значе́ние

  1. книжн. , устар. приноси́ть по́льзу кому-чему́-нибудь
    • Не зна́ешь ли, что друг есть поле́зен в печа́ли // И что мно́го по́льзовать мо́жет сове́т здра́вый, // Когда́ по нём поступа́ть не претя́т злы нра́вы?
    • мало ж по́льзует тебя́ звать хоть сы́ном ца́рским, // Бу́де в нра́вах с гну́сным ты не разнишься псарским.
    • Нау́ки по́льзуют везде́, // Среди́ наро́дов и в пусты́не, // В градском шу́му и наедине́, // В поко́е сла́дки и в труде́.
    • свойство коме́дии — издёвкой править нрав; // Смеши́ть и по́льзовать — прямо́й ея уста́в.
  2. употребля́ть, испо́льзовать, получа́ть по́льзу
    • Нет, не́мцы не би́ли её и не по́льзовали.←…→Червоноармейцы то́же не би́ли, но по́льзовать по́льзовали: на отши́бе живёт, кричи́ ― до кого́ докричишься?
  3. книжн. , устар. лечи́ть
    • Больно́го по́льзовал изве́стный врач.
    • Стал по́льзовать её тот // Ле́карь нанятой; // И ма́стью он глаза по всяк день ма́зал той.
    • Чтоб сей реко́й благодея́ний // Покры́лась вся ея страна́; // Я зрел бы цепь простра́нных зда́ний, // Где по́льзует больны́х она, // Где бе́дных пи́щей насыща́ет, // Где брошенных берёт сиро́т, // Где их леле́ет, возращает, // Где просвеща́ет свой наро́д.
