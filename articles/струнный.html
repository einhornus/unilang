стру́нный (27013)

[ˈstrunːɨj]
|||||

струна́ (struná) + -ный (-nyj)


|||||

Adjective

стру́нный • (strúnnyj) (no comparative)

  1. (relational) string (of a musical instrument )
    стру́нный музыка́льный инструме́нт ― strúnnyj muzykálʹnyj instrumént ― string instrument
    стру́нная про́волока ― strúnnaja próvoloka ― string wire (for a musical instrument, e.g. a guitar)
  2. (relational) string instrument
    стру́нный орке́стр ― strúnnyj orkéstr ― string orchestra
  3. (rare) sonorous, musical

Related terms

  • струна́ (struná), стру́нка (strúnka), стру́ночка (strúnočka)
  • семистру́нный (semistrúnnyj)
