крутизна́ (15777)


Pronunciation

  • IPA(key): [krʊtʲɪzˈna]

Noun

крутизна́ • (krutizná) f inan (genitive крутизны́, uncountable)

  1. steepness
  2. (colloquial) toughness, coolness, awesomeness

Значе́ние

  1. свойство по значе́нию прилага́тельного круто́й; характери́стика большо́го угла́ накло́на чего́-ли́бо
    • да вы ви́дите, кака́я крутизна́? тут косте́й не соберёшь!
  2. круто́й спуск, круто́е место
    • под ним весь в мы́ле конь лихой // Бесце́нной ма́сти, золото́й. — // Пито́мец ре́звый Карабаха // Прядёт ушьми, и, по́лный стра́ха, // Храпя́ коси́тся с крутизны́ // На пе́ну скачущей волны.
  3. жарг. характери́стика от́ круто́й
    • есть у меня́ оди́н знако́мый но́вый русский — крутизны́ немереной!
