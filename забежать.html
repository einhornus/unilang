забежа́ть -> to run in, call in, run off, (coll) drop in
Verb, perfective, intransitive, stress pattern: b

Я забегу́
Ты забежи́шь
Past: забежа́л/забежа́ла



Etymology: за- (za-) +‎ бежа́ть (bežátʹ)



Definitions (English):
  • to get (
    • в (v) + accusative or на (na) + accusative somewhere) by running or as if by running. Он забежа́л туда́ пе́ред рабо́той. On zabežál tudá péred rabótoj. He ran that way before work.
  • to run ahead (in anticipation), to run before someone,
    • забежа́ть вперёд ― zabežátʹ vperjód ― to run ahead to anticipate
  • (colloquial) to visit, to drop in (
    • к (k) on)


Definitions (Russian):
  • ненадо́лго посети́ть како́е-либо место
    • Он забежа́л туда́ пе́ред рабо́той
  • перен. передвигаясь бе́гом, обогна́ть или кого́-либо, оказа́ться впереди́ или сбо́ку
    • Ему́ удалось забежа́ть вперёд
