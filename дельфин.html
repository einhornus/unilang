дельфи́н (11380)


Etymology

Borrowed from German Delphin.

Pronunciation

  • IPA(key): [dʲɪlʲˈfʲin]
  • Audio
    		(file)

Noun

дельфи́н • (delʹfín) m anim (genitive дельфи́на, nominative plural дельфи́ны, genitive plural дельфи́нов)

  1. dolphin
    • 1797, Гаврила Державин, <cite>Рожде́ние красоты:
      Си́зы, ю́ные дельфи́ны, // Облелея табуном, // На свои́ её взя́в спины, // Мча́ли по пучи́не во́лн.
      Sízy, júnyje delʹfíny, // Obleléja tabunóm, // Na svoí jejó vzjav spíny, // Mčáli po pučíne voln.

Значе́ние

  1. зоол. кру́пное млекопита́ющее из отря́да китообразных, подотряда зубатых кито́в
    Там, вы́плыв из пучи́ны, // ‎Играют, резвя́тся дельфи́ны, // ‎И рыб стада сверка́ют чешуёй, // ‎И бле́щут чуды чре́ва белизно́й. Г. Р. Державин, «У́тро», 1800 г. (цита́та из Викитеки)
