инструмента́льный (18103)


Etymology

инструмент (instrumént) +‎ -альный (-álʹnyj)

Pronunciation

  • IPA(key): [ɪnstrʊmʲɪnˈtalʲnɨj]
  • Audio
    		(file)

Adjective

инструмента́льный • (instrumentálʹnyj)

  1. (music) instrumental
  2. (relational) tool, toolmaking
    инструмента́льное произво́дство ― instrumentálʹnoje proizvódstvo ― tool production
    инструмента́льная кладова́я ― instrumentálʹnaja kladovája ― tool crib
    инструмента́льная сталь ― instrumentálʹnaja stalʹ ― tool steel
    инструмента́льный ключ ― instrumentálʹnyj ključ ― tool wrench
    инструмента́льный цех ― instrumentálʹnyj cex ― tool shop
    инструмента́льная оши́бка ― instrumentálʹnaja ošíbka ― instrumental error
  3. (relational, computing) tool
    инструмента́льные средства ― instrumentálʹnyje srédstva ― toolbox
    инструмента́льная лине́йка ― instrumentálʹnaja linéjka ― toolbar
    инструмента́льное програ́ммное сре́дство ― instrumentálʹnoje prográmmnoje srédstvo ― software tool

Related terms

  • инструмент (instrumént)
  • инструментальист (instrumentalʹíst), инструментальщик (instrumentálʹščik)
  • инструментарий (instrumentárij)
  • инструментовать (instrumentovátʹ), инструментовка (instrumentóvka)
