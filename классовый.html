классовый -> class (political) (adj)
Adjective



Etymology: класс (klass) +‎ -овый (-ovyj)



Definitions (English):
  • (relational) class (social grouping, based on job, wealth, etc.)
    • кла́ссовое созна́ние ― klássovoje soznánije ― class consciousness, кла́ссовая борьба́́ ― klássovaja borʹbá ― class struggle


Definitions (Russian):
  • свя́занный, соотносящийся по значе́нию с существительным класс
    • Кла́ссовые противоре́чия. Кла́ссовая борьба́. Кла́ссовый враг
  • неол. соотве́тствующий идеоло́гии, интере́сам того́ или ино́го кла́сса
    • Кла́ссовая направленность поли́тики
