implementation (1475)


е́д. ч. мн. ч.
implementation implementations

im-ple-men-tation

Существи́тельное.

Ко́рень: --.

Значе́ние

  1. выполне́ние, исполне́ние, воплоще́ние, осуществле́ние, реализа́ция (де́йствие)
    Implementation of this plan begins next year. — Реализа́ция э́того пла́на начина́ется в сле́дующем го́ду.
  2. исполне́ние, воплоще́ние, реализа́ция (результа́т де́йствия)
    The implementation was less than perfect. — Исполне́ние бы́ло ме́нее соверше́нным.
  3. внедре́ние; ввод в де́йствие
