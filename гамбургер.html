га́мбургер (25121)


Pronunciation

  • IPA(key): [ˈɡambʊrɡʲɪr]

Noun

га́мбургер • (gámburger) masculine gender">m inan (genitive га́мбургера, nominative plural га́мбургеры, genitive plural га́мбургеров)

  1. hamburger (sandwich), beefburger, burger
  2. (graphical user interface) hamburger icon, hamburger button (an icon with three horizontal lines, indicating a menu in graphical user interfaces)

Usage notes

  • The graphical user interface sense is usually written with quotation marks, e.g. ико́нка «га́мбургер».

Related terms

  • чизбургер (čízburger)
