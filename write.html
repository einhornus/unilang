write (195)


Инфинитив write
3-е л. е́д. ч. writes
Прош. вр. wrote, writ
Прич. прош. вр. written, writ
Герундий writing

wri-te

Непра́вильный глаго́л.

Ко́рень: -write-.

Значе́ние

  1. пи́сать (соверша́ть де́йствие)
    Your son has been writing on the wall. — Ваш сыни́шка писал на стене.
  2. пи́сать (занима́ться ви́дом де́ятельности)
    My uncle writes newspaper articles for The Herald. — Мой дя́дюшка попи́сывает стате́йки для The Herald.
  3. брит. с предло́гом to, амер. без предло́га пи́сать (письма)
    брит. Please write to me when you get there. / амер. Please write me when you get there. — Пожа́луйста, (на)пиши́ мне, когда́ доберёшься.
  4. комп. пи́сать, запи́сывать (на диск и т. п.)
    The computer writes to the disk faster than it reads from it. — Этот компью́тер пи́шет на диск быстрее, чем считывает с него́ же.

Синонимы

  1. inscribe, scrawl, scribble
  2. author, pen
  3. send a letter
  4. save, store
