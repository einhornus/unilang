запозда́лый (13887)


Pronunciation[edit]

  • IPA(key): [zəpɐzˈdalɨj]

Adjective[edit]

запозда́лый • (zapozdályj)

  1. belated, late, tardy
  2. overdue
  3. later
  4. backward, back
  5. dilatory
  6. last, latter

Related terms[edit]

  • запа́здывать (zapázdyvatʹ), запозда́ть (zapozdátʹ)

Значе́ние[пра́вить]

  1. (о челове́ке или живо́тном) не яви́вшийся во́время ◆ То́, как зверь, она заво́ет, // То́ запла́чет, как дитя́, // То́ по кро́вле обветшалой // Вдруг соло́мой зашуми́т, // То́, как пу́тник запозда́лый, // К нам в око́шко застучи́т. А. С. Пушкин, «Зи́мний вечер» (цита́та из Викитеки)
  2. (о собы́тии) не случи́вшийся во́время ◆ Пусть в торжестве́ насме́шливом своём // Ум бесполе́зный сердца тре́пет // Угомонит, и тще́тных жа́лоб в нём // Удушит запозда́лый ле́пет. Е. А. Баратынский, «О́сень» (цита́та из Викитеки)

Сино́нимы[пра́вить]

  1. запозда́вший, задержа́вшийся
  2. запозда́вший
