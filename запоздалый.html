запозда́лый (13889)


Pronunciation

  • IPA(key): [zəpɐzˈdalɨj]

Adjective

запозда́лый • (zapozdályj)

  1. belated, late, tardy
  2. overdue
  3. later
  4. backward, back
  5. dilatory
  6. last, latter

Related terms

  • запа́здывать (zapázdyvatʹ), запозда́ть (zapozdátʹ)

Приста́вка: за-; ко́рень: -позд-; суффиксы: -а-л; оконча́ние: -ый [Тихонов, 1996].

Значе́ние

  1. (о челове́ке или живо́тном) не яви́вшийся во́время
    То, как зверь, она заво́ет, // То запла́чет, как дитя́, // То по кро́вле обветша́лой // Вдруг соло́мой зашуми́т, // То, как пу́тник запозда́лый, // К нам в око́шко застучи́т. А. С. Пушкин, «Зи́мний вечер» (цита́та из Викитеки)
  2. (о собы́тии) не случи́вшийся во́время
    Пусть в торжестве́ насме́шливом своём // Ум бесполе́зный сердца тре́пет // Угомонит, и тще́тных жа́лоб в нём // Удушит запозда́лый ле́пет. Е. А. Баратынский, «О́сень» (цита́та из Викитеки)

Сино́нимы

  1. запозда́вший, задержа́вшийся
  2. запозда́вший
